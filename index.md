```
ESC
Ctrl + [
```
## 基本編
### ノーマルモード
#### 最低限のこと
カーソル移動
```
hjkl
```
保存・終了
```
:w
:q
:q!
```
#### ノーマルモードで移動基本編
行頭へ
```
0
^
↑正規表現と同じ!
```
行末へ
```
$
↑正規表現と同じ!
```
#### ノーマルモードで編集基本編
一文字削除
```
x
```
行削除
```
dd
```
カーソル以降削除
```
d$
```
undo
```
u
```
redo
```
Ctrl + r
```
yank(ようはコピー。ただしyankは標準ではクリップボードには入らないので注意。クリップボードにいれるためのVimプラグインはある)
```
y
yy
↑行yank
y$
↑行末までyank
```
ペースト
```
p
```
ファイル内検索
```
/
```
ファイル跨ぎ検索
```
:grep
標準のvimgrepは遅い。rg等に差し替えると高速に動作していい感じになる。
ヒットした内容はQuickFixという仕組みで表示される。
hjklで移動可能。Enterで対象を開く。
QuickFixの履歴間の移動は:cold, :cnew
```
行内の置換
```
:s/new/old/gc
```
ファイル全体の置換
```
:%s/new/old/gc
```
選択範囲を置換
```
(範囲選択してから:すると自動的に:'<,'>になる)
:'<,'>s/new/old/gc
```
ファイル跨ぎの置換
```
vim-qfreplaceというVimプラグインを使おう
https://github.com/thinca/vim-qfreplace
```
#### ノーマルモードのその他基本的なこと
オプションのお試し
```
:set opt
例えば:set nu で行番号を出す
永続化したいものは設定ファイルに記述する
```
ヘルプの参照
```
:help
```
### ビジュアルモード
ビジュアルモードへ切り替え
```
(ノーマルモードで)
v
↑範囲選択をするモード。選択した範囲に対して後述するyankや置換操作など出来る
```
前回選択範囲を復活
```
gv
```
### ウィンドウ分割
水平分割
```
:sp
```
垂直分割
```
:vsp
```
ウィンドウ移動
```
Ctrl + w, hjkl
```
一個前のウィンドウ
```
Ctrl + w, Ctrl + w
```
Vim関係ないけどtmuxによるターミナル分割についても触れてもいいかも
## 応用編
### ノーマルモード
#### ノーマルモードの移動応用編
ファイルの先頭へ
```
gg
```
ファイルの末尾へ
```
G
```
一個前のカーソル位置へ
```
Ctrl + o
```
一個先のカーソル位置へ
```
Ctrl + i
```
対応するカッコへ
(,),[,],{, or }
```
%
```
#### ノーマルモードの編集応用編
大文字変換
```
(対象をビジュアルモードで選択してから)
U
```
小文字変換
```
(対象をビジュアルモードで選択してから)
u
```
